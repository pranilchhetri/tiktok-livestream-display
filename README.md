# Express.js and EJS

This repository showcase tiktok livestream data on web using node.js, express on backend and ejs as a view engine
### Endpoints

+ `/`
+ `/livedata`
+ `/teacher`

## Install

`npm install`

## Run

1. `npm start`
2. Open `http://localhost:8080` in a web browser.
